A real-time text translation application built with Angular.  
The app allows users to type text, select a target language, and receive instant translation with a smooth user experience.

## Features
- Real-time translation while typing using RxJS (debounce)
- Language selection via dropdown list
- Reactive Forms with validation
- HTTP Interceptor for handling loading states and API errors
- Global loading indicator
- Centralized error handling
- Clean and responsive dark UI
- Accessible form elements

## Technical Details
- Angular
- TypeScript
- RxJS
- Reactive Forms
- RESTful API integration
- HTTP Interceptors
- Environment-based configuration
- Service-based architecture (separation of concerns)
